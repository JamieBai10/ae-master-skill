## 核验策略

- 页面标题、菜单路径、支持格式和参数名称：优先具体功能页。
- “何时加入 / 是否修复 / 是否 Beta”：优先 Release Notes 和 What's new。
- 非英语界面与菜单翻译不一致：以英文菜单名、图标和路径为稳定定位；若用户指定中文版，再给中文对照。
- 旧项目：先查项目创建时版本和 expression engine；旧项目可能默认 Legacy ExtendScript，不能假定和新项目相同。
